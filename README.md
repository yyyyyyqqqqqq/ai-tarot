# 🔮 AI 塔罗牌占卜

AI 驱动的塔罗牌占卜应用。22 张大阿尔卡纳，三张牌阵（过去·现在·未来），Claude API 生成个性化解读。

## 使用方式

1. 打开网页
2. 点击右下角 ⚙️ 设置你的 Anthropic API Key
3. 输入你想问的问题
4. 翻开三张牌
5. 获取 AI 解读

## 部署

GitHub Pages: https://yyyyyyqqqqqq.github.io/ai-tarot/

## 技术

- 纯 HTML/CSS/JS，零依赖
- Claude API (Haiku)
- CSS 3D 翻牌动画
- localStorage 本地存储
